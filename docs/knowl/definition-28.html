Definition 2.2.1. Set operations defined via logic.

Let \(A \and B\) be sets.

  1. \(\displaystyle A \subseteq B \iff (x \in A) \to (x \in B)\)

  2. \(\displaystyle x \in A \cup B \iff x \in A \lor x \in B\)

  3. \(\displaystyle x \in A \cap B \iff x \in A \land x \in B\)

  4. \(\displaystyle x \in A \setminus B \iff x \in A \land x \not\in B\)

  5. \(A = B\) if and only if \(A \subseteq B\) and \(B \subseteq A\text{.}\)
