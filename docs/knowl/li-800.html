Item 1.

First, since \(\gcd(11,20)=1\text{,}\) there is a unique solution to this recurrence. Using either the Euclidean Algorithm or guessing-and-checking, notice that \(11\cdot11=121\equiv 1\pmod{20}\text{,}\) so \(11^{-1} \equiv 11\pmod{20}\text{.}\) So we have:

\begin{align*} 11x &\equiv 15\pmod{20} \\ 11\cdot11x &\equiv 11\cdot 15\pmod{20} \\ 1x &\equiv 165 \pmod{20} \\ x &\equiv 5 \pmod{20} \end{align*}
