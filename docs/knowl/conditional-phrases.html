Note 1.1.8.

There are many ways to phrase the conditional statement \(p \to q\text{.}\) Here are just a few common ones:
  • If \(p\text{,}\) then \(q\text{.}\)
  • \(p\) implies \(q\text{.}\)
  • \(p\) only if \(q\text{.}\)
  • \(p\) if sufficient for \(q\text{.}\)
  • \(q\) is necessary for \(p\text{.}\)
  • \(q\) if \(p\text{.}\)
  • \(q\) whenever \(p\text{.}\)
  • \(q\) unless \(\neg p\text{.}\)
