Theorem 3.3.6. The Euclidean Algorithm.

2300+ years old.
This is called the Euclidean Algorithm after Euclid of Alexandria because it was included in the book(s) of The Elements he wrote in around 300BCE. We donâ€™t know much about Euclid, but The Elements influenced all future Greek, Arab, and Western mathematics.
Let \(a\) and \(b\) be two positive integers where \(a \ge b\text{.}\) If we apply the division algorithm recursively so that
\begin{gather*} a = bq_1 + r_1 \text{ where } 0 \le r_1 \lt b\\ b = r_1q_2 + r_2 \text{ where } 0 \le r_2 \lt r_1\\ r_1 = r_2q_3 + r_3 \text{ where } 0 \le r_3 \lt r_2\\ r_2 = r_3q_4 + r_4 \text{ where } 0 \le r_4 \lt r_3\\ \dots \\ r_{n-2} = r_{n-1}q_{n-1} + r_n \text{ where } 0 \le r_n \lt r_{n-1}\\ r_{n-1} = r_{n}q_{n} + 0 \end{gather*}
Then \(\gcd(a,b) = r_n\text{,}\) the last non-zero remainder.
