Exercise 1.

Find the first five terms of the sequence \(\{a_n\}\text{,}\) beginning with \(n=0\text{:}\)

  1. \(a_n = 2^n + 3\)
  2. \(a_n = n^{n+1}\)
  3. \(a_n = \lfloor n/2\rfloor + \lceil n/2 \rceil\)
  4. \(a_n = 6a_{n-1},\hspace{1em} a_0 = 2\)
  5. \(a_n = a_{n-1} + 3 a_{n-2},\hspace{1em} a_0 = 1, a_1 = 2\)
